🔄 Tests de Non-Régression (TNR)
Pas un type de test à part entière, mais une stratégie d'exécution. Il s'agit de rejouer une suite de tests existants après chaque modification pour garantir qu'aucune régression n'a été introduite. Les TNR peuvent être composés de n'importe quel type de test ci-dessus.
